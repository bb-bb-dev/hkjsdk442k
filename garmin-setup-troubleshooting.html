TubeWitch setup flow

Complete these steps in order before troubleshooting individual symptoms.

  1. Use the SRAM AXS mobile app to assign the Front/Rear position and pressure target for each sensor.
  2. Install TubeWitch using the Garmin Connect IQ mobile app.
  3. Fully reboot the Garmin device and restart the Connect IQ mobile app.
  4. Add the TubeWitch data field to a data screen, leave the TubeWitch field visible, and rotate your wheels.
  5. Open TubeWitch settings in the Connect IQ mobile app or Garmin Express and enter the numeric part of each sensor ID.
  6. Confirm live front and rear pressure before the ride.

Settings do not save

ALERT: SAVING SETTINGS WITH THE CONNECT IQ MOBILE APP IS AFFECTED BY A BUG IN THE GARMIN IQ SYSTEM. MAKE SURE THE TUBEWITCH DATA FIELD IS VISIBLE ON YOUR GARMIN DEVICE WHEN CHANGING SETTINGS. IF SETTINGS STILL DO NOT SAVE, TRY GARMIN EXPRESS. VIEW CURRENT WORKAROUND

When using Connect IQ mobile app settings

  • Make sure the TubeWitch data field is visible on the Garmin device before changing settings.
  • Enter one sensor ID first and save immediately before entering the second.
  • Do not go back into the nested Sensor Set before tapping Save.
  • If Connect IQ still does not save reliably, use Garmin Express on a computer.
Open settings-save workaround

Nested settings can look like they reverted

If you back out of a Sensor Set or other nested settings group and then open it again before saving from the main settings screen, Garmin may show the old values.

Back out of the nested settings group, tap Save immediately, then reopen TubeWitch settings to confirm the values.

Fix the symptom you see

After the setup flow is complete, start with the section that matches what you see on the Garmin device.

No pressure data appears

  • Rotate the wheels to wake the sensors and look for a flashing LED.
  • Confirm each sensor is assigned Front or Rear in the SRAM AXS mobile app.
  • Confirm the numeric sensor IDs were saved in TubeWitch settings.
  • Acknowledge any Garmin open-connection message for each sensor.
  • Remove other TyreWiz or TPMS data fields, then reboot the Garmin device.
Open no-pressure troubleshooting

Pressure status or alerts look wrong

  • Check pressure targets and alert ranges in the SRAM AXS mobile app.
  • Confirm the correct front and rear sensor IDs are entered in TubeWitch.
  • Make sure high and low pressure alerts are enabled in TubeWitch settings.
  • Check the Alert Delay setting if an alert does not trigger when expected.
Open alert troubleshooting

FIT recording problems

Garmin limits how many Connect IQ data points can be written to an activity file. If fields record too many FIT data points, disable lower-priority recording options first.

Open FIT recording troubleshooting

App upgrade or install problems

If Garmin Connect IQ still shows Updating after a TubeWitch upgrade, sync the Garmin device, restart Garmin mobile apps, and reboot the device before troubleshooting settings.
