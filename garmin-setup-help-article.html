Garmin setup guide

TubeWitch setup and troubleshooting for Garmin

This article walks through the full TubeWitch setup path: prepare the sensors, install the Garmin data field, enter sensor IDs, confirm live pressure, and fix the common problems that can block setup.

Follow the sections in order. If something fails, stop at that section and use the linked troubleshooting page before changing unrelated settings.

Before you start

You need a compatible tire pressure sensor, a compatible Garmin device, the Garmin Connect IQ mobile app, and either the SRAM AXS mobile app or Garmin Express depending on what you are changing.

  • Use Compatible Tire Sensors to confirm the sensor family is supported.
  • Use the TubeWitch Garmin App Store page to confirm Garmin device compatibility.
  • Use the SRAM AXS mobile app to set the sensor Front/Rear position and pressure target.
  • Use TubeWitch settings to enter sensor IDs, labels, display options, alert options, and activity recording options.

Treat SRAM AXS and TubeWitch as two separate setup jobs. SRAM controls the sensor target and wheel position. TubeWitch controls how that sensor data is read and displayed on Garmin.

Recommended setup order

  1. Open the SRAM AXS mobile app and assign each sensor to Front or Rear.
  2. Set the pressure target and alert range for each sensor in SRAM AXS.
  3. Install TubeWitch from the Garmin Connect IQ mobile app.
  4. Fully reboot the Garmin device and restart the Connect IQ mobile app.
  5. Add TubeWitch to a Garmin data screen.
  6. Open the data screen so TubeWitch is loaded and visible.
  7. Open TubeWitch settings and enter the numeric part of each sensor ID.
  8. Save settings immediately.
  9. Rotate the wheels and confirm live pressure.

Leave the TubeWitch data field visible on your Garmin device when changing settings in the Connect IQ mobile app.

For the standard setup page and demo videos, open the TubeWitch setup guide.

Sensor ID entry

TubeWitch needs the sensor IDs so it knows which tire pressure sensors to read. For TyreWiz sensors, enter only the numeric part of each ID.

Example: if the sensor ID is HGL12345, enter 12345.

Use Garmin Connect IQ mobile app settings for normal setup. If settings do not save reliably, use Garmin Express on a computer.

If settings do not save

There is a Garmin Connect IQ settings issue that can prevent settings from saving reliably. Fix settings saves before troubleshooting sensor data.

Make sure TubeWitch is visible on the Garmin device before changing TubeWitch settings in the Connect IQ mobile app.

  1. Open the TubeWitch data screen on the Garmin device.
  2. Open TubeWitch settings in the Garmin Connect IQ mobile app.
  3. Enter one sensor ID first.
  4. Back out to the main settings screen and tap Save immediately.
  5. Do not go back into a nested Sensor Set before tapping Save.
  6. Reopen TubeWitch settings and confirm the values are still there.

If that still fails, use the Garmin Express workaround on the settings not saving troubleshooting section.

Confirm live pressure

After settings are saved, rotate the wheels and confirm front and rear pressure before the ride.

  • Pressure values should appear for the enabled sensors.
  • The values should update after the wheels rotate.
  • The pressure status should match the target and alert range set in SRAM AXS.
  • If Garmin shows an open-connection message, acknowledge it for each sensor.

If no pressure data appears

Work from the sensor outward before changing advanced settings.

  • Rotate the wheels and check that the sensor wakes up.
  • Confirm the sensor can be reached in the SRAM AXS mobile app.
  • Confirm each sensor is assigned Front or Rear in SRAM AXS.
  • Confirm the numeric sensor ID was saved in TubeWitch settings.
  • Remove other TyreWiz or TPMS data fields, then reboot the Garmin device.
  • If a sensor is not assigned Front/Rear or is a generic ANT+ TPMS sensor, try TubeWitch Labs -> Multi-Sensor Compatibility Mode.

For the full checklist, open No Pressure Data.

If pressure status or alerts look wrong

TubeWitch follows the pressure target and alert range stored in the sensor. Those values are set in the SRAM AXS mobile app.

  • Confirm the target pressure and alert range in SRAM AXS.
  • Confirm the front and rear sensor IDs are not swapped in TubeWitch settings.
  • Make sure high and low pressure alerts are enabled in TubeWitch settings.
  • Check Alert Delay if alerts do not trigger when expected.
  • Remember that vibration alerts are watch-only, and popup alerts require supported newer devices.

For the full alert checklist, open Wrong Pressure Status/Alerts.

If FIT recording causes IQ restarts

Garmin limits how many custom FIT data points Connect IQ apps can record at once. If Garmin shows the IQ logo or a data field restarts during recording, the combined recording load may be too high.

Disable lower-priority Activity Recording options in TubeWitch settings, then reboot Garmin before testing again.

For the full field count list, open FIT Recording Problems.

If an app upgrade appears stuck

If Garmin Connect IQ still shows Updating after a TubeWitch upgrade, sync the Garmin device, restart Garmin mobile apps, and reboot the Garmin device.

Open App Upgrade Problems for the focused upgrade steps.

Contact TubeWitch support

Use the Garmin Connect IQ mobile app. Open the TubeWitch Garmin App Store page, tap Open in App, scroll to the bottom, then tap Contact Developer.

Include your email address to receive a reply. Also include your Garmin device model and the exact step that is failing.
