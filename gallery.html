Watch support

TubeWitch also fits Garmin watches

Compatible watches can show the same pressure data with a layout tuned for smaller screens and device-specific constraints.

Popup, tone, and vibration behavior depend on the device, while the data field settings still control labels, colors, decimal precision, and layout behavior.

What changes between layouts

Labels

Short or explicit labels

Use label and delimiter settings to keep compact layouts cleaner or make them more descriptive.

Display

Orientation and compatibility mode

Layout orientation and display mode help TubeWitch fit devices with different screen shapes and rendering support.

Alert flow

Popup, tone, vibration, and timing

Alert output and timing can be adjusted so warnings stay visible without turning into nuisance duplicate alarms.

Profiles

Multiple bikes and wheelsets

Named sensor sets, Touch Data Field actions, and Auto Switch behavior make the same layouts practical across several bikes.
